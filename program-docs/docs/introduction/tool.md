## FAQ

#### Q: 迁移工具是否可以实现多次迁移？ 

A：当前迁移工具仅支持一次迁移

 

#### Q: 迁移工具都有哪些内容不能迁移的？ 

A：登陆、支付、自定义组件及wxs不能迁移，其他内容可以迁移。

 

#### Q: 要使用webview如何操作？ 

A：webview中要打开的页面需要设置白名单，到用户后台，进入开发设置中，将网站域名加入到配置中
